- Servidor de Web que suporta php (Apache,Abyss,Xitami,IIS...)

- 32 MB de RAM
- 2 MB de espaço livre

3 - Permissões
    *No Windows:
		Você precisa dar todas as permissões à pasta texto (leitura,escrita, execução)e para o arquivo tpl1.html;
	*No Unix:
		Dê chmod 777 no diretório texto e para o arquivo tpl1.html;
	
4 - Configure o chat:
		abra o arquivo config.php num editor

		*Mudando o nome do chat =
			 procure a seguinte linha
				$cfg['EC_CHAT'] = 'Enygmata Chat';
			substitua Enygmata Chat pelo nome desejado
		*Mudando a senha do chat = 
			procure a seguinte linha
				$cfg['EC_SENHA'] = '1234';
			substitua 1234 pela senha desejada.

5 - Pronto você já pode utilizar o chat
