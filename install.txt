3 - Permissões
	*No Windows:
		Você precisa dar todas as permissões à pasta texto e seus arquivos (leitura,escrita, execução);
	*No Unix:
		Dê chmod 777 no diretório texto e para seus arquivos;
	
4 - Configure o chat:
		-Inicie seu servidor(se não estiver iniciado) e entre no endereço :
			http://enderecodoseuservidor/caminho/enygmatachat/adm.php

		-O usuário é admin
		-a senha é 1234
		
		-vá para a sessão Configuração/Configuration
		significado das configurações:
			EC_TEMPLATE            = Layout do chat
			EC_ADMINISTRADOR       = login do administrador
			EC_AUTENTICACAO_IMAGEM = autenticação com imagem; 0 desativado,1 ativado
			EC_BLOQUEAR            = bloqueio do chat; 0 desativar, 1 ativar
			EC_BW_SEARCH           = lista de palavras ruins(separadas por virgulas)
			EC_EMAIL  EC_LINGUAGEM = idioma do chat
            EC_MODO_DEPURACAO      = ativar modo de depuração; 0 desativar, 1 ativar, 2 ativar extendido
			EC_NOME_CHAT           = nome do chat
			EC_NUM_SALAS           = numero de salas
			EC_PREFIX              = prefixo das salas
			EC_SENHA               = senha da administração 
			EC_TAM_TEXTO           = tamanho maximo do nick e do texto



5 - Pronto você já pode utilizar o chat
